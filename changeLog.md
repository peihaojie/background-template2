# 更新日志

### 【0.0.3】 - 2024-04-17

#### 新增
- 【filters】 增加 eventHandlers 事件抛出
- 【filters】 增加 changeProp 支持修改组件双向绑定的数据
- 【filters】 增加 changeSearch 属性，控制组件搜索的触发
